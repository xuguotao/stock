## 当前规则

- 所有项目文档使用中文叙述。
- 代码标识、命令、API 路径、表名、字段名和文件路径保留原文。
- 新 spec、plan、review 必须基于当前代码、当前数据和当前用户目标重新形成。
- 不自动继承归档文档里的旧结论。
