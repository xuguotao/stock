# 紫金矿业监控报告买入判断规则

## 使用原则

监控报告是买入条件过滤器，不是直接买卖信号。优先看紫金矿业趋势、黄金趋势、铜趋势和产量兑现是否共振。

## 核心判断

### 1. 价格趋势先止弱

如果报告显示：

```text
紫金矿业：弱
黄金：弱
铜：中性或弱
```

不急着买入，继续观察。更合理的买入前提是：

- 紫金矿业从 `弱` 变成 `中性`，说明跌势放缓或重新接近 60 日均线。
- 紫金矿业进一步变成 `强`，说明重新站上 20 日和 60 日均线。
- 黄金或铜至少有一个不弱；更好的情况是铜和黄金都不弱。

### 2. 产量兑现不能恶化

重点看矿产金和矿产铜，锂是增量故事但不是当前主逻辑。

- 矿产金、矿产铜从 `观察` 变成 `达标`，买入条件改善。
- 碳酸锂当量从 `落后` 变成 `观察`，增量预期改善。
- 如果矿产铜转为 `落后`，即使铜价强，也要降低买入意愿。

## 操作分层

| 报告状态 | 操作倾向 |
|---|---|
| 紫金弱 + 黄金弱 + 铜中性/弱 | 不急，继续观察 |
| 紫金中性 + 铜中性/强 + 黄金不再弱 | 可以小仓试探 |
| 紫金强 + 铜强 + 黄金中性/强 | 更适合分批买入 |
| 紫金强但金铜都转弱 | 小心假突破 |
| 矿产金或矿产铜转落后 | 降低买入意愿 |

## 规则化条件

### 买入观察

```text
紫金矿业状态从 弱 -> 中性
且 铜不是弱
且 矿产铜没有落后
```

### 小仓买入

```text
紫金矿业为 中性 或 强
且 铜为 中性 或 强
且 黄金不再弱
且 矿产金、矿产铜至少都是 观察
```

### 加仓

```text
紫金矿业为 强
且 铜或黄金至少一个为 强
且 矿产金、矿产铜有一个或两个变成 达标
```

### 回避

```text
紫金矿业为 弱
且 黄金、铜同时为 弱
或 矿产铜变成 落后
```

## 当前解读模板

如果报告显示“紫金矿业弱、黄金弱、铜中性”，结论偏向：

```text
不适合追买，适合等待趋势修复后的右侧确认。
```

更实用的执行口径：

```text
不要在股价弱、黄金弱时因为觉得便宜就买；
等紫金矿业先从弱转中性，再看铜价和产量是否配合。
```
